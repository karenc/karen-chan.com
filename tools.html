Tools

  • YouTube Downloader
    A ruby on rails site that lets you search on youtube and download the videos in different formats.
  • Live departure board (UK train)
    A python cgi which screen scrape the nationalrail site, with simplified html and compact display optimized for mobile browsers.
  • E-mail Tester
    A site that lets you create free, disposable email addresses (a very simplified version of Mailinator) Written in Python with Google AppEngine. It was used mainly for testing emails in development.
  • Browser Details
    A perl cgi that prints out details about your browser.
  • Translate Chinese characters into pinyin
    A python cgi that uses the baidu dictionary to translate Chinese paragraphs into pinyin, works with both Traditional Chinese and Simplified Chinese. Useful for learning Chinese (Mandarin).
  • Irssi script to email messages to you when you are away (Download)
    A perl irssi script that when you set yourself as away, sends an email to you when someone mentions your name on IRC or when you get a private message.
  • Irssi script to show the page title of URLs (Download)
    A perl irssi script that shows you the page title of URLs on IRC in the status window. Requires libwww-perl.
